UNISOC Poweroff Driver

This IP block in ths unisoc SoC that only deals with different ways to power the system down.

Required properties:
- compatible:
  -"sprd,sc2721-poweroff" for SC2721 poweroff
  -"sprd,sc2730-poweroff" for SC2730 poweroff
  -"sprd,sc2731-poweroff" for SC2731 poweroff
- reg: Specifies the physical address of the poweroff register.

Example:
poweroff@1820 {
	compatible = "sprd,sc27xx-poweroff", "sprd,sc2730-poweroff";
	reg = <0x1820>;
};
